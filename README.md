##  Datasets Used

1. **ny_property_sales.csv**  
   - **Content:** Property sales records in New York City  
   - **Key Columns:**  
     - `Borough` (Region)  
     - `Sale Price`  
     - `Sale Date`  
     - `Building Class`  
     - `Square Feet`  

2. **sales2017_raw.csv**  
   - **Content:** Company retail sales transactions  
   - **Key Columns:**  
     - `Region` (City/Area)  
     - `Date` (Transaction Date)  
     - `Sales` (Revenue)  
     - `Profit`  
     - `Product`  

>  Both datasets are linked via regional information to enable meaningful analysis.
